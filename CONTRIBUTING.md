# Contributing to Phase-Slip-Sampler

We welcome contributions from the community!

## How to Contribute
1.  Fork the repository.
2.  Create a new branch for your feature (`git checkout -b feature/AmazingFeature`).
3.  Commit your changes.
4.  Open a Pull Request.

## Research Goals
We are specifically looking for benchmarks on:
*   Larger models (Llama-3, Mistral).
*   Different heat levels for different tasks (Coding vs Creative Writing).
